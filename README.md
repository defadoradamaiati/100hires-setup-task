# 100Hires Setup Task

## Overview

This repository documents the setup process completed as part of the 100Hires application process. The goal was to install the required tools, create a GitHub repository, document the process, and record any issues encountered during setup.

## Tools Installed

* Git for Windows
* Cursor IDE
* GitHub Account

## Tools Requested but Not Successfully Installed

* Claude Code
* Codex

## Steps Completed

1. Installed Git for Windows.
2. Verified Git installation through Command Prompt.
3. Installed Cursor IDE.
4. Created a GitHub account.
5. Created a public GitHub repository.
6. Created and updated the README.md file.
7. Documented the setup process and troubleshooting efforts.
8. Published the repository to GitHub.

## Issues Encountered

### Issue 1: Git was not installed

When attempting to use Git commands in Command Prompt, the system returned an error indicating that Git was not recognized.

#### Resolution

* Downloaded Git for Windows.
* Installed Git using the default settings.
* Verified the installation successfully after reopening Command Prompt.

### Issue 2: Claude Code could not be installed

The instructions required installing Claude Code in Cursor. After exploring the available Plugins section and searching for Claude Code, the plugin was not available.

#### Actions Taken

* Opened the Plugins section in Cursor.
* Searched for "Claude Code".
* Checked available plugin options.
* Researched installation guidance.
* Documented the issue and proceeded with the remaining required tasks.

### Issue 3: Codex could not be installed

The instructions required installing Codex in Cursor. After searching within Cursor, the plugin was not available for installation.

#### Actions Taken

* Opened the Plugins section in Cursor.
* Searched for "Codex".
* Reviewed available plugin options.
* Documented the issue and continued with the remaining setup steps.
