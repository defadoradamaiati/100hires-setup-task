# 100Hires Setup Task

## Overview

This repository documents the setup process completed as part of the 100Hires application process. The objective was to install the required tools, create a GitHub repository, document the process, and record any issues encountered along the way.

---

## Tools Installed

* Git for Windows
* Cursor IDE
* GitHub Account

---

## Steps Completed

1. Installed Git for Windows.
2. Verified Git installation through Command Prompt.
3. Installed and opened Cursor IDE.
4. Created a GitHub account.
5. Created a public GitHub repository.
6. Added a README.md file.
7. Documented the setup process and troubleshooting steps.
8. Published the repository to GitHub.

---

## Issues Encountered

### Issue 1: Git was not installed

When attempting to use Git commands in Command Prompt, the system returned an error indicating that Git was not recognized.

#### Resolution

* Downloaded Git for Windows.
* Installed Git using the default installation settings.
* Verified the installation successfully after restarting Command Prompt.

### Issue 2: Claude Code plugin could not be located

The instructions referenced a Claude Code add-on for Cursor. After exploring the available plugins and searching within Cursor, the plugin was not available in the installed version.

#### Resolution

* Searched the plugin marketplace within Cursor.
* Reviewed available plugin options.
* Documented the issue and continued with the remaining required setup steps.
